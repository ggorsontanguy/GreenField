How to use these poc:

1) Unzip the file on OUR Desktop

2) Have Java 1.7 right installed and configurated

3) Have Maven right installed and configurated

4) Run: mvn clean install  on the root of the project.
	If BUILD SUCCESSFULL our are ready.

5) To be easy to rollback when you try some thing: use GIT

git init

git add .

git commit -m "init project"
